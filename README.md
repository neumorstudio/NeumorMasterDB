# Neumor Directory (Next.js)

Aplicacion web en Next.js (App Router) para buscar y filtrar negocios/servicios desde Supabase.

## Proyecto activo

- App: `web/`

## Ejecutar local

```bash
cd web
npm install
npm run dev
```

## Validaciones

```bash
npm run lint
npm run typecheck
npm test
npm run build
```

## Variables de entorno

Configura `web/.env.local` usando `web/.env.example`.
